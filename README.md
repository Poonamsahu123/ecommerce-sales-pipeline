 E-Commerce Sales Pipeline & Dashboard

 Project Overview

This project demonstrates an end-to-end data pipeline using Azure and Power BI.

Tools Used

* Azure Blob Storage
* Azure Data Factory
* Azure SQL Database
* Power BI

Data Pipeline

CSV → Azure Blob → Data Factory → SQL Database → Power BI

 Dashboard Features

* Total Sales KPI
* Sales by City
* Sales by Product Line
* Sales Trend
* Filters for analysis

 Insights

* Highest sales in Yangon
* Food & Beverages is top category
* Customer behavior varies by type

Conclusion

This project shows how cloud-based tools can be used for data processing and visualization.
